1. Executive Summary

BrainScope AI is an end-to-end, browser-based medical imaging analysis system that automatically detects and classifies seven brain pathologies from CT/MRI scan images. The system combines a cloud-hosted object detection model (Roboflow RF-DETR), a GitHub Pages static frontend, and a multi-agent automation pipeline.

Key capabilities:

  • Multi-disease detection with bounding box localisation
  • DICOM (.dcm) native support — reads raw hospital scan files directly in-browser
  • AI-generated radiology report with urgency classification and specialist referral
  • Self-improving pipeline — agents automate dataset collection, training, and deployment

CLAHE Preprocessing

Contrast Limited Adaptive Histogram Equalization (CLAHE) is applied to all training images before upload. Unlike global histogram equalization, CLAHE operates on local image tiles, preventing over-amplification of noise in uniform regions — critical for CT scans where tissue contrast is subtle.

clahe = cv2.createCLAHE(clipLimit=4.0, tileGridSize=(8, 8))
enhanced = clahe.apply(grayscale_image)

DICOM Window/Level Rendering

CT scans store pixel values in Hounsfield Units (HU). The brain window (WC=40, WW=80) is applied by default if not found in the DICOM header, mapping HU values to 0-255 grayscale:

val = clamp(((HU - (WC - WW/2)) / WW) × 255, 0, 255)

🔍 DataHunter

Searches 7+ known Roboflow Universe repositories for brain imaging datasets. Prioritises underrepresented classes (bleeding, ischemia). Outputs datahunter_results.json with image counts and accessibility status.

Run: python agent_datahunter.py [--download]

🏋️ AutoTrainer

Compares new trained model against current best (tracked in autotrainer_state.json). Tests accuracy on local test images. If new model beats old by any margin, updates frontend/index.html, commits, and pushes to GitHub Pages automatically.

Run: python agent_autotrainer.py [status|deploy|test N|check-v6]

🔬 Inspector

Audits dataset quality: (1) counts images per class to detect imbalance, (2) calculates Laplacian variance for blur detection (threshold <50), (3) uses current model to flag images where prediction disagrees with label at ≥50% confidence.

Run: python agent_inspector.py [full|balance|quality|mislabeled]

📋 ReportWriter (Browser Agent)

Embedded JavaScript in frontend/index.html. Triggered by "Generate Report" button after analysis. Maps predictions to a disease database (DISEASE_DB) containing urgency level, specialist recommendation, and clinical action. Outputs a printable HTML report.

Output: HTML modal with urgency banner, findings table, specialist referral list, disclaimer

git clone https://github.com/chokdee255120-dot/brainscope-ai.git
cd brainscope-ai
git config user.email "totoolclaude@gmail.com"
git config user.name "chokdee255120-dot"

Step 2: (Optional) Run local backend

start_backend.bat           # Windows — starts FastAPI on http://localhost:8000
# OR:
cd backend && uvicorn main:app --reload

Step 3: Deploy frontend change

# Edit frontend/index.html, then:
Copy-Item frontend/index.html index.html    # PowerShell
git add index.html frontend/index.html
git commit -m "your message"
git push origin main
# GitHub Pages rebuilds automatically (~2 min)

Step 4: Train new model version

# Quick: generate version from existing 21,806 images and train
python gen_v6_now.py

# Full: CLAHE preprocessing + upload + generate + train (overnight)
python v6_build.py

# Monitor and auto-deploy when training completes
python watch_v6.py

Step 5: Deploy best model

python agent_autotrainer.py status    # Check current best
python agent_autotrainer.py deploy    # Compare + auto-deploy

Step 6: Run inspection

python agent_inspector.py full        # Full quality report
python agent_datahunter.py --download # Find + download new data

Path to 98%+ mAP

Based on published research (YOLO11 with attention: 99.3% mAP@0.5, RF-DETR Large: 99.5% on meningioma), achieving 98%+ requires:

  1. RF-DETR Large (requires paid Roboflow plan — accurate training mode)
  2. Balanced dataset — at least 1,000 images per class for all 7 classes
  3. CLAHE preprocessing on all training images (v7, currently uploading)
  4. Full augmentation — mosaic, mixup, rotation, flip, brightness, noise
  5. Clean labels — remove label0/1/2 pollution via Roboflow class remapping
